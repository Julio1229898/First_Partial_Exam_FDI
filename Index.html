First Partial Exam Interactive Design Fundamentals
Frontend
The frontend is the stetic and the beauty of the page, when we are doing front end what we are looking for is to make the page the most pleasant and enjoyable possible for the user, for us to make this happen we use different things like html, css, javascript, etc. so that the experience of the user on our web page could be the best.
Backend
The Backend is everything behind the web page so it can work, it's where all the data or information goes so it can be processed so it can be sent to the user later, it also helps with the connection with the server and the Database.
Client
The client is a person, company, organization, other that is willing to pay for something (on this case the webpage) and is not necessary the user of your product (webpage)
End User
The end user is the one that is going to use or access our product (webpage), when we are designing our product we should think on the end-user or user.

HTML
HTML started as the official language for the World Wide Web and was conceived in 1990, it was a product of the Standard Generalized Markup Language (SGML), SGML was a complex and technical specification describing markup languages, HTML was originally created for those who were not specialized in SGML to publish and exchange different kinds of documents it facilitated the exchange of documents by incorporing the ability to link documents electronically using hyperlinks, people started to see that HTML was easy and it started to proliferate.
CSS
As more people started using HTML, the demand for more design capabilities grew, this would help developers to control how their web documents looked, in October 1994, Hakon Wium Lie release the first draft of "Cascading HTML Style Sheets" and Brent Bos was building Argo, a browser that had style sheets which could be used by other markup languages, in all, nine different style sheets proposal where submitted and the discussions on the www-style mailing list influenced the developments of CSS, after some years in August 1996 Microsoft Internet Explorer became the first browser to support CSS, then Netscape followed suit in supporting CSS but they also implemented an alternative Javascript Style Sheets, which were never fully completed.
